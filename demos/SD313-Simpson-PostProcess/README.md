# From Posterior to Post Processing: Getting More from Your Bayesian Model

## Files:

1. postprocess.pdf : PDF file of the slides for this presentation
2. posprocess_run.sas : SAS script with all code for this presentation

## Additional information
See also:
1. [SAS43110-2020 - Incorporating Auxiliary Information into Your Model Using Bayesian Methods in SAS® Econometrics](https://github.com/sascommunities/sas-global-forum-2020/tree/master/papers/4311-2020-Simpson)
   Paper describing how to select priors in Bayesian inference. Both default priors, and informative priors are considered.

2. [SD308 - Incorporating Auxiliary Information into Your Model Using Bayesian Methods in SAS® Econometrics](https://github.com/sascommunities/sas-global-forum-2020/tree/master/demos/SD308-Simpson-AuxiliaryInfo)
   Super Demo showing how to automatically construct the default priors described in the above paper. Includes SAS code you can adapt for your own problems.
